## 简介

本仓库是南京大学软件学院《互联网计算》课程的改进计划的内部仓库，视情况发布后会公开。

本仓库的文档组织使用了VuePress的形式，以方便版本控制以及发布Web版本。实际线下导出时仍然使用PDF这样的形式。PDF内尽量使用PDF自带的标签实现目录跳转，最后如需导出纸质版，则使用手动目录的方式导出。后期也可能采取GitBook这样的形式导出为真正的书的形式。不过反正Markdown文档里面其实也就具体的Markdown内容，实际上放哪里并没有什么区别。
